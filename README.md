### Prerequisites

- Rust (latest stable version)
- GTK4 development libraries
- Cargo

### Building

```bash
# Clone the repository
git clone https://github.com/st93642/VDownloader.git
cd VDownloader

# Build the project
cargo build --release

# Run the application
cargo run
```
